**CalculadoraGUI**

**Descripción**
Este proyecto es una calculadora sencilla con interfaz gráfica.  
Al ejecutar el programa se abre una ventana donde puedes introducir dos números y pulsar el botón **Calcular**, a continuación, el programa muestra la **suma** de ambos valores.

Actualmente la calculadora solo realiza sumas, pero está en desarrollo para incorporar más operaciones aritméticas como resta, multiplicación y división.

**Tecnologías utilizadas**
Python y la librería Tkinter.

**Cómo ejecutar el proyecto**
1. Clona este repositorio.
2. Abre la carpeta del proyecto en Visual Studio Code.
3. Asegúrate de tener Python instalado.
4. Ejecuta el archivo principal (por ejemplo `suma_gui.py`) desde Visual Studio Code.

**Estructura del proyecto**
Este proyecto solo tiene un fichero llamado "suma_gui.py".
